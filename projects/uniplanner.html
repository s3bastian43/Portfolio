Uniplanner .

Uniplanner is a Student Planner and Team Collaboration Web Application developed for my dissertation project. The website goal is to simplify the life of a university student by offering, in one application, a set of features in which the user can keep track of his course in a more efficient manner.

03. The Result

In April 2018, after approximately four months of designing and development, the web application was set live (available at www.uniplanner.co.uk). However, as there are still some sections which are unfisnished (e.g. Blog), the website was not yet promoted.
