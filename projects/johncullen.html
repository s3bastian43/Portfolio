John Cullen Gardens .

John Cullen Gardens is a multiple award-winning company based in Lincolnshire which specialises in growing plants for the pollinators, plants for scent and plants for taste. Initially, John Cullen had a simple brochure website built using a WYSIWYG Web Builder, but in October 2017 they reached out the Easitill Web Team for a website redesign.

Visit Website

01. My Role

Building the John Cullen Gardens website was my first task when I started working for Easitill. I was asked then to provide a solution for a company who wishes to engage with a variety of users on a new nature-inspired website. Therefore, I designed two prototypes and built an eCommerce website in WordPress.

02. Design

As the clients were both plants experts they were really excited about colours which can immediately make the user feel like he is shopping through their actual nursery. However, they were undecided and initially believed that a website loaded with a lot of green elements may be too much.
Therefore, I designed a set of two prototypes in Adobe InDesign:
  • The first set which was the one chosen for the final version represents a fresh, natural look which uses green as the primary colour and amber as the accent colour to indicate that the website is about plants and pollinators.
  • The second set represents an earthy, forest-inspired look which uses brown as the primary colour and green as the accent colour.

Once this stage was finished I moved on and started to build the website in WordPress.

03. Technology

The website uses a variety of plugins and a custom Avada Theme. The eCommerce part of the website uses a custom Easitill plugin which renders the pages with the client products.
I made use of HTML, CSS and JavaScript to create, stylise and modify the elements. The performance and the search engine optimisation were considered throughout the whole development process as I personally consider that these two play a vital role for every online business.

04. The Result

The website was finished by the end of October; however, it was only set live at the beginning of 2018 because a payment integration had to be made on the back-end part of the website.

05. Summary

John Cullen Gardens was one of those projects which gave me a huge freedom in term of design as the client didn’t had a definite brand and wasn’t sure in which direction they should had to. The feedback was positive so I believe that I provided a practical website and therefore a solution which fulfilled the client needs.
